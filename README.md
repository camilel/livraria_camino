# Livraria Camino
Online bookstore system developed as final graduation project.

## Overview
This project is a web application that simulates an online bookstore.  
Main features include:
- Customer registration and login
- Browsing books by categories
- Shopping cart functionality
- Order and payment processing

## Technologies
- PHP (back-end)
- MySQL (database)
- CSS (styling)
- HTML (structure)

## Project Structure
- PHP files (add_cliente.php, cadastro.php, etc.)
- CSS files (stylesheets)
- bd_camino.sql
- imagens/  # folder containing images

## Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/livraria_camino.git
   
2. Set up a local server (XAMPP, WAMP, or similar).

3. Import the bd_camino.sql file into MySQL.

4. Update database credentials in conexao.php.

5. Access the system in your browser at http://localhost/livraria_camino.
